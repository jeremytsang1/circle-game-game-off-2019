* Description
Try to grab the green balls before being attacked by the red balls.

* Requirements
- Requires:
  |----------+----------------|
  | Name     | Version Tested |
  |----------+----------------|
  | Python 3 |          3.6.9 |
  | PyGame   |          1.9.6 |
  |----------+----------------|

* Instructions
** Starting Game
- Run game by either running either of the following:
  1) Running as python program
     #+BEGIN_SRC bash
       $ python main.py
     #+END_SRC
  2) Running as executable script
     #+BEGIN_SRC bash
       $ chmod u+x main.py  # Make file executable (only needs to be done once)
       $ ./main.py          # Run the game
     #+END_SRC

** Controls
|-----+-----------------------|
| KEY | DESCRIPTION           |
|-----+-----------------------|
| ↑   | jump to outer orbits  |
| ↓   | jump to inner orbits  |
| ←   | move counterclockwise |
| →   | move clockwise        |
|-----+-----------------------|
